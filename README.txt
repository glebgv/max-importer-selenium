# Telegram to MAX web transfer script

Скрипт для переноса сообщений (текст, фото, аудио, видео) из экспортированного чата Telegram (например, "Избранное") в чат "Избранное" веб-версии MAX (https://web.max.ru/).  
Отправка осуществляется через буфер обмена (Ctrl+V) для любых типов медиа, без открытия диалоговых окон.

## Возможности

- Парсинг HTML-файлов экспорта Telegram (формат, создаваемый десктоп-версией Telegram).
- Отправка текстовых сообщений с сохранением переносов строк.
- Отправка медиафайлов (фото, аудио, видео) через буфер обмена (изображения копируются как изображения, остальные файлы – как список файлов).
- Добавление подписей к медиа (текст после медиа).
- Автоматическая пауза после каждых N сообщений (настраивается в коде).
- Статистика по отправленным сообщениям.

## Требования

- Python 3.10+
- Установленный браузер Chrome
- ChromeDriver, совместимый с версией Chrome (скачать с [официального сайта](https://sites.google.com/chromium.org/driver/))
- Экспорт чата Telegram (сообщения и медиа) в формате HTML

## Подготовка экспорта Telegram

1. Откройте Telegram Desktop.
2. Перейдите в нужный чат (например, "Избранное").
3. Нажмите на три точки (меню) → "Export chat history".
4. В настройках экспорта:
   - Оставьте только нужные типы данных: текст, фото, видео, аудио.
   - Формат: HTML.
   - Укажите папку для экспорта, например `C:\Users\Admin\Downloads\Telegram Desktop\ChatExport_2026-03-12 (1)`.
5. Запустите экспорт. После завершения в указанной папке появятся файлы `messages.html` (или несколько) и папки `photos`, `files` с медиа.

## Установка зависимостей

Создайте виртуальное окружение (опционально) и установите необходимые пакеты:

```bash
pip install selenium beautifulsoup4
```

# Настройка скрипта

1. Скачайте скрипт max_importer.py (или переименуйте его соответственно).

2. Откройте файл в текстовом редакторе и при необходимости измените переменные в начале:

```
EXPORT_PATH = r"C:\Users\Admin\Downloads\Telegram Desktop\ChatExport_2026-03-12 (1)"  # путь к папке экспорта
BATCH_SIZE = 100   # количество сообщений в одном батче
BATCH_DELAY = 5    # пауза между батчами в секундах
```

# Запуск

1. Убедитесь, что Chrome закрыт.
2. Откройте терминал (командную строку) в папке со скриптом.
3. Выполните команду:

```
python3 max_importer.py
```

4. Скрипт автоматически откроет окно браузера с сайтом MAX.
5. Отсканируйте QR-код для входа в свою учётную запись MAX.
6. После входа перейдите в чат "Избранное" (выберите его в интерфейсе).
7. Вернитесь в консоль, где запущен скрипт, и нажмите Enter.
8. Скрипт начнёт отправлять сообщения из экспорта, отображая прогресс в консоли.

# Примечания

- Скрипт копирует изображения в буфер обмена как изображения, а остальные файлы – как список файлов. Это позволяет вставлять их в MAX через Ctrl+V без открытия диалогов.

- Если файл не найден по указанному в HTML пути, скрипт ищет его в подпапках photos и files.

- Сообщения отправляются последовательно, между сообщениями пауза 2 секунды (настраивается в коде).

- При возникновении ошибок (например, не найден файл) скрипт пропускает элемент и продолжает работу.

- Для прерывания работы используйте Ctrl+C.
